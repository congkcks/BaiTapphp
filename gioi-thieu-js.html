Học JavaScript cơ bản

JavaScript là ngôn ngữ lập trình giúp tạo ra các trang web động, tương tác và hiện đại. Bạn có thể bắt đầu học JavaScript từ các chủ đề sau:

  • Biến, kiểu dữ liệu, toán tử
  • Câu lệnh điều kiện, vòng lặp
  • Hàm, đối tượng
  • DOM: thao tác với nội dung trang web
  • Sự kiện (event)
  • AJAX, JSON, API

Ví dụ JavaScript cơ bản

<button onclick="myFunction()">Click Me!</button>
<script>
function myFunction() {
  alert('Xin chào JavaScript!');
}
</script>
